# VimTutor

1. Starting Vimtutor.

```sh
$ vim test.txt
```

2. Starting vimtutor - Inside vim window enter following:

```vim
:help tutor
```

3. For using the GUI version of Vim, use following:

```sh
$ vimtutor -g
```
