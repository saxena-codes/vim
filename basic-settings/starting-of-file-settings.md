# Starting of File Settings

## Code

- Helps in following functionalities:
	- Set file encoding.
	- Show number line.
	- `nocompatible` for plugin support.

- To be entered inside `.vimrc`, at the starting of file:

```
" Set encoding
set encoding=utf-8

" Setting line number
set number

" Required by various plugins to work
set nocompatible
```
