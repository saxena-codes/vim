# onedark.vim

Overall theme for vim. Personal preference, can be changed to any other theme.

## Code

```vim
call plug#begin()

" Theme
Plug 'joshdick/onedark.vim'

call plug#end()

" Setting colorscheme to onedark
colorscheme onedark
```
