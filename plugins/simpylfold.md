# SimpylFold

A vim plugin to fold `Python` nested code automatically.

## Code

1. `vim-plug` code to add to `.vimrc` file:

```
call plug#begin()

Plug 'tmhedberg/SimpylFold'

call plug#end()
```

2. After adding above code and saving, open vim and run `:PlugInstall`.
