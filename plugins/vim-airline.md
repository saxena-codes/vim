# vim-airline Plugin

```
" Imported Plugins
call plug#begin()

Plug 'vim-airline/vim-airline'
Plug 'vim-airline/vim-airline-themes'

call plug#end()

" Setting vim-airline Theme
let g:airline_theme='base16-spacemacs'

```
